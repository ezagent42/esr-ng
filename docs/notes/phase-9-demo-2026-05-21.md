## How to reproduce the demo

```bash
# Reset DB
rm -f ~/.ezagent/default/db/*.db

# Migrate + seed
MIX_ENV=dev mix ecto.create
MIX_ENV=dev mix ecto.migrate

# Set admin password (post-migration boot-time seed creates admin row with NULL password)
MIX_ENV=dev mix ezagent.user.set_password "entity://user/system/admin" --password "8bdemo"

# Start phx
MIX_ENV=dev mix phx.server

# Browse
open http://localhost:10042/login  # admin / 8bdemo / workspace=system
```

For headless screenshots (as captured for this doc), see `take_screenshots.py` in `/tmp/phase9-demo/` (developer scratch — not committed).
